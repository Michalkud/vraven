# VRAVEN — Vrastislav Vender

Personal-brand site studies for **Vrastislav Vender**. One idea, many executions:
a warm beam of light from the dark illuminates the name, with **VRA** (VRAstislav) +
**VEN** (VENder) lit so they read **VRAVEN** — the rest kept in shadow.

**▶ Gallery of every variant: [gallery.html](./gallery.html)**

- `vraven-site.html` — full one-page redesign (WebGL god-rays + GSAP/Lenis motion)
- `vraven-minimal.html` — abstract / minimal cut
- `index.html` — original CSS-light version (also the no-WebGL fallback)
- `variants/` — earlier cinematic studies
- `cave/` — cave-light placement studies
